# 🌐 DecentraLeziaHub - Web3, AI & Digital Skills Learning Platform

A free, open-source learning hub for Web3, Crypto, Blockchain, AI, and digital career transitions.

## 🎯 Mission
Bridge the gap between Web2 and Web3 careers through bite-sized courses, resources, and community support.

## 📚 What You'll Learn

### Core Learning Paths
- **Web3 Fundamentals** - Smart contracts, DeFi, NFTs basics
- **Crypto & Blockchain** - How crypto actually works
- **AI Tools & Skills** - Practical AI applications
- **Digital Marketing** - Web3-native marketing strategies
- **Data Entry Skills** - Remote work foundations
- **Career Transition** - Web2 → Web3 roadmap

### Additional Resources
- Job opportunities
- Airdrop guides
- Freelance opportunities
- DAO contributor paths
- Tools & communities

## 🚀 Getting Started

### For Learners
1. Visit the [Learning Hub](./docs/01-web3-basics/intro.md)
2. Pick your learning path
3. Complete bite-sized lessons
4. Apply knowledge with practical exercises

### For Contributors
See [CONTRIBUTING.md](./CONTRIBUTING.md)

## 🛠 Tech Stack

- **Framework**: Docusaurus 3.x (React-based documentation framework)
- **Styling**: Tailwind CSS + custom components
- **Hosting**: GitHub Pages (free)
- **Content**: Markdown + MDX
- **Search**: Algolia (free tier)

## 📖 Content Philosophy

- **Modular**: Short, focused lessons (5-15 min reads)
- **Accessible**: No jargon, beginner-friendly
- **Free**: All content free and open
- **Practical**: Real examples, not theory
- **Community-driven**: Built by the community
